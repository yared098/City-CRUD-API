# City CRUD API

A **modularized Express.js REST API** for managing city data using an **in-memory array**.  
This project demonstrates **CRUD operations** (Create, Read, Update, Delete) with clean folder structure and `.env` configuration for the server port.

---

## Features

- Modularized project structure
- CRUD operations for `City` resource:
  - Get all cities
  - Get city by ID
  - Add a new city
  - Update existing city
  - Delete a city
- Uses `.env` to configure server port
- In-memory storage (no database needed)
- Ready to extend for database integration
